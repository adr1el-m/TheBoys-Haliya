## Slide 4: How It Works

### Patient Journey
1. **Describe Symptoms** → Natural language input (EN/FIL)
2. **AI Analysis** → Safety-layered LLM assessment + deterministic rules + evidence ledger
3. **Get Results** → Urgency score + recommendations
4. **Take Action** → Book appointment or follow guidance

### Technology Stack
- **Frontend**: Next.js 16 + TypeScript + Tailwind
- **Backend**: Node.js + Express + PostgreSQL
- **AI**: Groq (LLaMA 3.3 70B) for fast inference, wrapped with auditability and safety overrides
- **Infrastructure**: Vercel + Railway + Neon
